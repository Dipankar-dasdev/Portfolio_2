# 🌐 Personal Portfolio

A modern and responsive personal portfolio website built using **HTML, CSS, and JavaScript**. This project showcases my skills, projects, and contact information through a clean and interactive user interface.

## 🚀 Live Demo

🔗 https://dipankar-dasdev.github.io/Portfolio_2/

## ✨ Features

- Modern and responsive design
- Smooth scrolling navigation
- Animated mobile navigation menu
- Interactive hover effects and transitions
- Project showcase with Live Demo & GitHub links
- Technology badges for each project
- Contact section
- Social media links
- Clean and organized code structure

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Boxicons
